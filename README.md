# mutantdevs.com

This is the source for https://mutantdevs.com 

it is built with next.js

## Develop

```
npm run dev
```

## Export Static Site

```
npm run build && npm run export
```

You will then find your site available under `./out`

To serve it locally:

```
npx http-server out
```

Then you should find it under:

http://localhost:8080

## Current Deployments
